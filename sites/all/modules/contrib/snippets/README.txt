The Snippets module allows you to store and display code snippets on Drupal sites.

## Installation

- Snippets module depends on Features (features), Field collection (field_collection), Geshi filter (geshifilter) module.
- Just enable Snippets module.
- Create a snippet by going to Content -> "Add content" -> "Code snippets".


## Drush download

drush dl snippets features field_collection geshifilter views libraries
